# Retail Store SQL Data Analysis Project

## Project Overview
This project analyzes a Retail Store dataset using SQL to understand customer behavior, product performance, and sales trends.

The dataset contains multiple tables related to customers, orders, products, payments, and reviews. Using SQL queries, I explored business questions to generate meaningful insights.

## Database Tables
The database consists of six tables:

- Customers
- Orders
- Order_Items
- Products
- Payments
- Product_Reviews

## Key SQL Analysis

Some of the business questions answered in this project:

- Retrieve customer names and emails for marketing
- Display products within a mid-range price range (₹2000 – ₹5000)
- List products sorted by price
- Calculate total value per order item
- Identify out-of-stock products
- Calculate total number of orders
- Calculate total revenue
- Find top customers by total spending
- Analyze products sold by category
- Analyze orders by day
- Identify most used payment methods

## Key Insights

- Total Revenue: ₹6,960,973
- Total Orders: 400
- Electronics is the highest selling category
- Highest order days: Friday, Wednesday, Monday
- Debit Card and Credit Card are the most used payment methods
- Top customers spent between ₹300000 – ₹500000
